# Environment Variables

- `RRFM_SEED`: Seed for random number generator (no seed used by default)
- `RRFM_SEGMENT_THRESHOLDS`: Comma-separated list of segment thresholds for each dimension (default: 200,200,200,200,200)
- `RRFM_BOX_SIZE`: Comma-separated list of box sizes for each dimension (default: 40,40)
- `RRFM_RECORD`: Whether to record the motion (default: 1)
- `RRFM_RECOVER_FROM_STAND_STILL`: Whether to recover from stand still (default: 1)
- `RRFM_ALWAYS_RANDOM_ROTATION`: Whether to always perform random rotation, instead of segment directed rotation (default: 0)
